## 📞 常见问题

### Q: 启动失败怎么办？
A: 检查Node.js是否安装，端口3000是否被占用

### Q: 压缩失败怎么办？
A: 检查源文件夹路径、文件权限、图片格式

### Q: 界面显示异常？
A: 清除浏览器缓存，检查网络连接

### Q: 如何停止服务器？
A: 在启动器窗口按任意键即可停止
